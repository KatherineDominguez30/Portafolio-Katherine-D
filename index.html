Experiencia Profesional

Encriptador de Texto

Proyecto realizado como Challenge del curso " Oracle ONE & Alura Latam ", consiste en crear un encriptador que reciba untexto y al presionar el botón encriptar mostrara el texto encriptado en el cuadro lateral, luego este texto encriptado puede ser copiado para introducirlo en el input principal y desencriptarlo. El diseño nos fue entregado en Figma y le realice algunas modificaciones.

Proyecto realizado con HTML, CSS y JavaScript.

Juego Ahorcado

Proyecto realizado como Challenge del curso " Oracle ONE & Alura Latam ", este es el popular juego del ahorcado donde el jugador debe adivinar la palabra oculta, por cada letra que digite si es correcta la imprimirá en pantalla sobre un guion bajo, en caso contrario se irá dibujando la horca por cada letra errónea.

El juego creado con HTML, CSS y JavaScript.
